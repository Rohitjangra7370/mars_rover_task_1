# Mars Rover — Task 1

Initial task work for Team Deimos Mars Rover project. Foundation for the autonomous navigation and control stack used in the University Rover Challenge (URC).

## Context

Team Deimos is IIT Mandi's entry in the University Rover Challenge (URC) — an international Mars rover competition. This repository captures early-stage task implementation for the rover's software stack.

## Stack

- ROS2
- Python / C++

## Team Deimos

- Competition: University Rover Challenge (URC)
- Achievement: Qualified ARC 2026 Finals (Anatolian Rover Challenge, international)
- Role: Rohit Jangra — Software Lead
